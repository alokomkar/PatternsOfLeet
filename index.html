1. The Two Pointers Pattern

🧠 Secret Clues to Spot This Pattern:

• Data layout is completely linear (Array, String, Linked List).
• Inputs are explicitly Sorted or sorting provides instant optimization.
• Target requires inspecting matching ends concurrently to find a pair or threshold.

Two Sum II - Input Array Is Sorted EASY

"Given a 1-indexed sorted array, find two numbers that add up to a target."

Pattern Hack: The sorted guarantee allows converging endpoints directly without allocating any extra storage space.

fun twoSum(numbers: IntArray, target: Int): IntArray {
    var left = 0
    var right = numbers.size - 1

    while (left < right) {
        val currentSum = numbers[left] + numbers[right]
        if (currentSum == target) {
            return intArrayOf(left + 1, right + 1)
        } else if (currentSum < target) {
            left++
        } else {
            right--
        }
    }
    return intArrayOf()
}
Container With Most Water MEDIUM

"Find two lines that form a container storing the maximum amount of water."

Pattern Hack: Maximize width by starting at the absolute margins, then progressively drop the smaller wall constraint.

fun maxArea(height: IntArray): Int {
    var left = 0
    var right = height.size - 1
    var maxWater = 0

    while (left < right) {
        val width = right - left
        val currentHeight = minOf(height[left], height[right])
        maxWater = maxOf(maxWater, width * currentHeight)

        if (height[left] < height[right]) {
            left++
        } else {
            right--
        }
    }
    return maxWater
}

2. Sliding Window

🧠 Secret Clues to Spot This Pattern:

• Demands sequential subsegments ("substring" or "subarray").
• Looking for dynamic optimization expressions like "longest" or "shortest" subsets.

Maximum Average Subarray I EASY

"Find a contiguous subarray of fixed length K with the maximum average value."

Pattern Hack: Keeps frame size exactly at K elements. Drops trailing values dynamically as incoming updates arrive.

fun findMaxAverage(nums: IntArray, k: Int): Double {
    var runningSum = 0.0
    for (i in 0 until k) runningSum += nums[i]
    var maxSum = runningSum

    for (i in k until nums.size) {
        runningSum += nums[i] - nums[i - k]
        maxSum = maxOf(maxSum, runningSum)
    }
    return maxSum / k
}
Longest Substring Without Repeating Characters MEDIUM

"Find the length of the longest substring without repeating characters."

Pattern Hack: Slide right to grow scope, and shrink left boundaries immediately whenever duplicates break unique map rules.

fun lengthOfLongestSubstring(s: String): Int {
    var left = 0
    var maxLength = 0
    val uniqueChars = HashSet()

    for (right in s.indices) {
        val incomingChar = s[right]
        while (uniqueChars.contains(incomingChar)) {
            uniqueChars.remove(s[left])
            left++
        }
        uniqueChars.add(incomingChar)
        maxLength = maxOf(maxLength, right - left + 1)
    }
    return maxLength
}

3. Prefix Sum

🧠 Secret Clues to Spot This Pattern:

• Needs multiple arbitrary slice calculations over stationary targets.
• Handles sequence tracking parameters containing negative intervals where standard window conditions break.

Range Sum Query - Immutable EASY

"Calculate sum of elements of nums between indices left and right inclusive."

Pattern Hack: Converts linear iteration costs down to single subtraction queries by storing running total counts.

class NumArray(nums: IntArray) {
    private val prefix = IntArray(nums.size + 1)

    init {
        for (i in nums.indices) {
            prefix[i + 1] = prefix[i] + nums[i]
        }
    }

    fun sumRange(left: Int, right: Int): Int {
        return prefix[right + 1] - prefix[left]
    }
}
Subarray Sum Equals K MEDIUM

"Return total number of continuous subarrays whose sum equals K."

Pattern Hack: Combines a historical tracking Map with rolling prefix valuations to look backwards for target segments.

fun subarraySum(nums: IntArray, k: Int): Int {
    var count = 0
    var currentSum = 0
    val prefixMap = HashMap()
    prefixMap[0] = 1

    for (num in nums) {
        currentSum += num
        if (prefixMap.containsKey(currentSum - k)) {
            count += prefixMap[currentSum - k]!!
        }
        prefixMap[currentSum] = prefixMap.getOrDefault(currentSum, 0) + 1
    }
    return count
}
